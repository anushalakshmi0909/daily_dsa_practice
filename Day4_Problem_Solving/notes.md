## Observations 📝

Both problems involve breaking a larger problem into **smaller repeated subproblems**.

Example:

Fibonacci(5)
= Fibonacci(4) + Fibonacci(3)

Fibonacci(4)
= Fibonacci(3) + Fibonacci(2)

Notice that **Fibonacci(3) is computed multiple times**.

Dynamic Programming avoids this repetition.
